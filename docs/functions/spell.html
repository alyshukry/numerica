    Function spell

    • spell(n: number, options?: Options): string

      Converts a number into its English words representation.

      This function handles both integer and decimal parts of the number. It supports optional formatting for hyphens between tens and ones, inclusion of "and" in hundreds, and a custom separator between words.

      Parameters

      • n: number

        The number to convert. Can be an integer or float.

      • Optionaloptions: Options = {}

        Optional settings for formatting.

        • hyphens

          Whether to include hyphens between tens and ones (e.g., "twenty-one").

        • and

          Whether to include "and" after hundreds (e.g., "one hundred and twenty").

        • separator

          The string used to separate words in the output.

      Returns string

      The English words representation of the number.

      Example

      spell(123) 
      // Returns: "one hundred twenty three"

      Example

      spell(123, { and: true }) 
      // Returns: "one hundred and twenty three"

      Example

      spell(45.67, { hyphens: true }) 
      // Returns: "forty-five point six seven"

      Example

      spell(1001, { separator: "-" }) 
      // Returns: "one-thousand-one"
